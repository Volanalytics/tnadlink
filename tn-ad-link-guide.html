Introduction to Banner Types

TN Ad Link supports a variety of banner types to help you create effective advertising campaigns tailored to Tennessee markets. Understanding the different options available will help you choose the best format for your marketing goals.

Did you know? Studies show that using the right banner type for your audience can increase click-through rates by up to 30%. TN Ad Link provides Tennessee-specific performance data to help you optimize your campaigns.

1. Image Banners

Image banners are the most common type of advertisement in TN Ad Link. They consist of a static image (JPG, PNG, or GIF) that links to your destination URL when clicked.

Creating an Image Banner

To create an image banner in TN Ad Link:

  1. Navigate to your campaign and click "Add new banner"
  2. Select "Image Banner" as the banner type
  3. Upload your image file (ensure it meets the size requirements)
  4. Enter the destination URL
  5. Save your banner

Key Benefits

  • Simple to create and implement
  • Works well across all Tennessee publisher sites
  • Fast loading times
  • Compatible with all browsers

Pro Tip for Tennessee Advertisers

Use images that resonate with Tennessee audiences, such as local landmarks, state symbols, or imagery that reflects regional interests like country music, outdoor activities, or local events.

2. HTML Banners

HTML banners allow for more complex and interactive advertisements using HTML, CSS, and JavaScript. This banner type offers greater creative flexibility than static images.

Creating an HTML Banner

To create an HTML banner in TN Ad Link:

  1. Navigate to your campaign and click "Add new banner"
  2. Select "HTML Banner" as the banner type
  3. Enter your HTML code in the editor
  4. Add any required assets (images, scripts)
  5. Save your banner

Sample HTML Code

<div style="width:300px; height:250px; border:1px solid #ccc; font-family:Arial;">
  <h3 style="background:#0057e7; color:white; padding:10px; margin:0;">Visit Tennessee!</h3>
  <div style="padding:10px;">
    <p>Experience the beauty of the Smoky Mountains!</p>
    <img src="mountains.jpg" width="280" height="120" alt="Smoky Mountains">
    <button style="background:#d31f1f; color:white; border:none; padding:8px 15px; 
            margin-top:10px; cursor:pointer;">Learn More</button>
  </div>
</div>

Key Benefits

  • Greater design flexibility
  • Interactive elements to engage users
  • Ability to incorporate Tennessee-themed animations
  • Can include multiple call-to-action elements
Important: HTML banners must follow TN Ad Link guidelines to ensure they work properly across all publisher sites. Avoid excessive JavaScript that might impact page performance.

3. Text Ads

Text advertisements consist of a headline, description, and display URL that links to your destination. They are simple, effective, and work well in content-heavy environments.

Creating a Text Ad

To create a text ad in TN Ad Link:

  1. Navigate to your campaign and click "Add new banner"
  2. Select "Text Ad" as the banner type
  3. Enter your headline, description, and display URL
  4. Enter the destination URL
  5. Save your text ad

Key Benefits

  • Quick and easy to create
  • Load instantly on all devices
  • Blend naturally with editorial content
  • Effective for local service advertising

Tennessee Text Ad Tips

Include local area codes (615, 423, 865, 901, 931) or specific Tennessee regions in your headline to increase relevance to local audiences.

4. Rich Media Banners

Rich media banners incorporate advanced features like animation, video, and interactive elements to create a more engaging user experience.

Creating a Rich Media Banner

To create a rich media banner in TN Ad Link:

  1. Navigate to your campaign and click "Add new banner"
  2. Select "Rich Media" as the banner type
  3. Upload your rich media file or HTML5 package
  4. Configure interaction settings
  5. Enter the destination URL
  6. Save your banner

Key Benefits

  • Higher engagement rates than static banners
  • Ability to showcase multiple products or features
  • Interactive elements encourage user participation
  • Can include video content for greater impact
Note: Rich media banners may have larger file sizes and specific requirements. Contact our Tennessee support team at support@tnadlink.com for assistance with rich media campaigns.

Best Practices for Tennessee Advertisers

1. Localize Your Creative

Use imagery and messaging that resonates with Tennessee audiences. Include local landmarks, references, or dialect when appropriate.

2. Size Optimization

Keep file sizes as small as possible without compromising quality to ensure fast loading times, especially in rural Tennessee areas with varying internet speeds.

3. Clear Call to Action

Every advertisement should have a clear, compelling call to action that tells Tennessee users exactly what you want them to do.

4. Mobile Optimization

With over 65% of Tennessee internet users browsing on mobile devices, ensure your banners are mobile-friendly.

5. A/B Testing

Create multiple versions of your banners and test them to see which performs best with Tennessee audiences.
